# Homebrew Tap for Tori

This is the official [Homebrew](https://brew.sh) tap for **Tori** — a desktop meeting capture and summarisation app.

## Installation

```bash
brew install --cask sodiqogundairo/tori/tori
```

Or add the tap first:

```bash
brew tap sodiqogundairo/tori
brew install --cask tori
```

## Updating

```bash
brew upgrade --cask tori
```

## Uninstalling

```bash
brew uninstall --cask tori
```

## About

This tap is automatically updated when a new version of Tori is released.

For more information, visit [tori-neon.vercel.app](https://tori-neon.vercel.app).
